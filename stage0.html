The World Wide Web
HTML is the most common document type used on the web.
Your browser makes requests via the internet to servers using a http protcol.
The server responds with files that are displayed on the browser.

HTML Basics
HTML is made up of text content, markup, references to documents, and links to other pages.
HTML markup is made up of tags which usually precede (opening tag) and succeed (closing tag) the contents.
The entire entity composed of tags and contents are called elements.

Types of Tags
The "b" tag is used to make text bold .
The "em" tag is used to make text emphasized or italicized.
The "a" tag is used to anchor text or link them to other pages.
The "p" tag is used to create paragraphs or blocks of text; which is also why it's called a block element.
The "br" tag is usde to create a line break, but does not create a box around the text; this makes it an inline element.
The "span" tag contain its contents while allowing specific styles or attributes to it; it is an inline element.

The "div" tag, like the "span" tag, simply contains its contents; however, this is a block element.

Document Structure
The first line should contain the document type, which is simply HTML when using html5.
The "html" tag opens the document and everything contained in the document should be between the opening and closing html tags.
The "head" tag defines metadata for the webpage which includes the title, as well as any references to javascript or css files.
The "body" tag contains the actual contents of the document.

Other Notes
